## Features

- Generate QR codes from arXiv ID or URL
- PNG output format
- Various styles (plain, preprint-red, mono)
- Customizable center labels and captions
- Shareable URLs with query parameters
- Download functionality

## Development

```bash
npm install
npm run dev
```
